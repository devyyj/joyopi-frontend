- 웹 서비스 기능 추가
- 이름 : 비밀 투표
- 설명 : 오프라인으로 함께 있는 2명 이상의 사용자가 웹 페이지에 접속하여 실시간 투표를 진행하고 결과를 확인할 수 있는 서비스
- 기능
  - 비밀 투표방
    - 생성
      - 방 리스트를 출력하지 않기 때문에 사용자는 어떤 방이 존재하는 지 알수 없다
      - 즉, 방은 비공개 이다
      - 하지만 현재 몇개의 방이 존재하는지는 알수 있다
      - 방 이름은 숫자 4자리로 임의로 정해지며 방 이름을 알고 있으면 방에 참여할 수 있다
      - 다른 사람이 쉽게 참여할 수 있도록 QR 코드가 생성된다
      - QR 코드를 스캔하면 비밀번호 없이 접속할 수 있다
      - 누구나 생성할 수 있다
      - 방장은 생성한 사용자
    - 삭제
      - 방에 참여자가 아무도 없으면 방은 자동으로 삭제된다
      - 방장이 방을 나가면 참여자중에 임의로 한명이 방장이 된다
    - 참여
      - 방이름 (숫자 4자리)을 알고 있으면 방에 참여할 수 있다
    - 나가기
      - 나가기 버튼을 눌러 방을 나갈수 있다
  - 투표
    - 방에 접속한 순서대로 '투표시작'버튼을 누를 수 있다
    - 자기 차례가 아니면 투표시작 버튼은 누를 수 없다
    - 투표시작 버튼을 누르면 10초간 카운트다운이 진행되고 이때 모든 사람들은 '투표하기' 버튼을 눌러서 투표할 수 있다
    - 투표
    - 투표하지 않을 수도 있다
    - 카운트다운이 끝나면 전체 인원중에 몇명이 투표했는 지 알수 있다
    - 하지만 이때 누가 투표했는 지는 알수 없다
    - 오직 투표 결과만 공유된다
    - 투표가 끝나면 다음 사람에게 차례가 돌아가며 '투표시작' 버튼이 활성화 된다
    - '넘기기' 버튼으로 차례를 넘길 수 있다
  - 접속자
    - 현재 접속자를 확인할 수 있으며 몇번째 차례인지 확인할 수 있다
    - 재접속한 사람은 차례가 가장 마지막으로 밀려난다
